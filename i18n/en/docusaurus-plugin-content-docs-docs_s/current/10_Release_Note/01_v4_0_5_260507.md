# RDKS100_LNX_SDK_V4.0.5_20260507

## **Release Date:** 2026-05-07

## 1. Key Information

**RDK S100 V4.0.5_20260507** release includes:

- **Linux Kernel:** 6.1.158-rt58
- **Bootloader:** U-Boot 2022.04
- **Root File System:** Based on Ubuntu 22.04 (Jammy), built with Multistrap
- **Supported Host Systems:**
  - Build environment: Ubuntu 20.04 (x64)
  - Flashing environment: Ubuntu 18.04 / 20.04 / 22.04 (x64)
- **Cross-compilation Toolchain:** arm-gnu-toolchain-11.3.rel1 (aarch64)
- **Other Components:** D-Robotics drivers, necessary firmware, toolchains, and utilities

## 2. New Features

None

## 3. Bug Fixes

- Fixed support for RDK S100 V1P1 hardware

## 4. Known Issues

### 4.1 Software Known Issues

- RDK Linux
  - When no HDMI cable is connected, the desktop system (GDM3) on RDK S100 defaults to fallback to X11 rendering. The S100 does not currently support GPU acceleration for X11, and falling back results in CPU rendering, leading to increased CPU usage.
- MCU1 SDK
  - MCU1 currently does not support lock-step unlocking
